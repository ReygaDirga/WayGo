## Description

WayGo is a travel planner web application with the tagline “Pack your bags, let’s go!”, designed to help users plan their trips in a more convenient, practical, and organized way. Currently, trip planning is often complex, as users need to rely on multiple separate applications to search for destinations, create itineraries, manage budgets, and coordinate with travel companions.

WayGo addresses this problem by providing an all-in-one platform that integrates these features into a single system, enabling users to plan their journeys more efficiently and comfortably.

## Features

- Smart Itinerary Generator
- Destination Recommendations
- Travel Budget Estimation
- Interactive Map Integration
- Social & Collaboration Feature
- Saved Trips
- Blog & Travel Inspiration

## Tech Stack

- **Backend:** Laravel  
- **Frontend:** Blade + Tailwind CSS  
- **Database:** MySQL
